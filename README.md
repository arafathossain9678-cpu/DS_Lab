# 📘 Data Structures – Code Repository

Welcome to the **Data Structures Code Repository** 👩‍🏫👨‍🏫  

This repository contains well-structured and classroom-oriented implementations of fundamental **Data Structures** concepts. The codes are designed to help students understand core principles, logic building, and practical implementation.

---

## 🎯 Purpose of This Repository

As a teacher, I created this repository to:

- Provide clear and easy-to-understand implementations
- Support classroom teaching and lab practice
- Help students prepare for exams and interviews
- Strengthen problem-solving and programming skills
- Offer reference material for assignments and projects

---

## 📚 Topics Covered

The repository includes implementations of:

### 🔹 1. Arrays
- Traversal  
- Insertion  
- Deletion  
- Searching (Linear & Binary Search)  

### 🔹 2. Linked Lists
- Singly Linked List  
- Doubly Linked List  
- Circular Linked List  
- Insertion & Deletion operations  

### 🔹 3. Stack
- Stack using Array  
- Stack using Linked List  
- Applications (Expression Evaluation, Parenthesis Checking)  

### 🔹 4. Queue
- Simple Queue  
- Circular Queue  
- Priority Queue  
- Queue using Linked List  

### 🔹 5. Recursion
- Basic recursive problems  
- Factorial  
- Fibonacci  
- Tower of Hanoi  

### 🔹 6. Trees
- Binary Tree  
- Binary Search Tree (BST)  
- Tree Traversals (Inorder, Preorder, Postorder)  

### 🔹 7. Graphs
- Graph Representation (Adjacency Matrix/List)  
- BFS (Breadth First Search)  
- DFS (Depth First Search)  

### 🔹 8. Sorting Algorithms
- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  

### 🔹 9. Searching Algorithms
- Linear Search  
- Binary Search

## ▶️ How to Use This Repository

1. Clone the repository:

```bash
git clone https://github.com/FForhad/DS_Lab.git
```

2. Navigate to the required topic folder.

3. Compile and run the program:

For C:
```bash
gcc filename.c
./a.out
```

For C++:
```bash
g++ filename.cpp
./a.out
```
